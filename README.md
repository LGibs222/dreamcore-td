# Dreamcore TD

A browser-based tower defense game with a dreamcore space aesthetic. Built by Lenwood + Xavier as a father-son coding project.

## Play locally

Open `index.html` in a browser. That's it — no build step.

If things look weird (e.g. modules, fetches), run a local server instead:

```bash
# Python 3
python -m http.server 8000
# then visit http://localhost:8000
```

## Milestones

See `_Index.md` for the full roadmap. Current: **M4 — towers shoot projectiles; enemies take damage and die.**

## Stack

HTML + vanilla JavaScript + Canvas. No frameworks, no build tools.
